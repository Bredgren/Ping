Feedback instructions

This game is currently a work in progress and feedback of any kind is welcome. Probably the easiest thing to do would be to send me a message over facebook. If you found the link in a message I wrote than you can respond to that, or you can find my page here and just send me a message directly.

Notes

  • I'm pretty sure the game only works in Chrome right now. It doesn't work at all in IE and the display is weird in FireFox.
  • I know it is possible to get the ball stuck behind the paddle, but I made it so that it won't continue to score while it is stuck.

Control panel

There should be a control panel in the upper right which you can use to modify a few of the game's behaviors if you want to. If you decide to do so then here are things to know (otherwise you may press the 'h' key to completely hide them).

  • All parameters except for "ball_type" take effect immediately.
  • Changes to "ball_type" will take effect each round.
  • I'm mostly interested in your experience with "ball_type" 0 (the circle ball) but feel free to play around with the others you want.
  • Play with the numbers and tell me what you liked best. If you know why you liked it then that helps too.

Thank you for trying my game.


- Brandon Edgren
